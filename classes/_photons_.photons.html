Class Photons

Photons class. A state of many photons, each with with dimensions: x, y, direction, polarization

see

{@link @Dimension.position}, {@link @Dimension.direction}, {@link @Dimension.polarization} Designed so that it will work with https://github.com/stared/quantum-game-2 board.
